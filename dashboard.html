⑥ 사용한 공식 (PDF 핵심 개념)

모비율 95% 신뢰구간 (p44)
p̂ ± 1.96 · √(p̂·q̂ / n)
오차범위 (MoE)
MoE = 1.96 · √(p̂(1-p̂)/n)
최대 MoE — p̂=0.5 일 때 (p51)
1.96 · √(0.25/800) = 3.46 %p 1.96 · √(0.25/503) = 4.37 %p 1.96 · √(0.25/1000)= 3.10 %p
두 후보 차이의 분산 (다항분포)
Var(p̂A - p̂B) = (pA + pB - (pA-pB)²) / n
검정통계량 / p-value (p59)
Z = (p̂A - p̂B) / SE_Δ p-value = 1 - Φ(Z)
중심극한정리 (p28)
σ(X̄) = σ / √n → 표본 4배 = 폭 1/2
왜 1.96 인가? (PDF p15 표준정규분포) 표준정규분포에서 양쪽 꼬리 합 5%를 잘라낼 때 |Z|=1.96. 표에서 P(0 ≤ Z ≤ 1.96) = 0.475 → 양쪽 합쳐 0.95. "욕먹지 않을 만큼"의 95% 신뢰수준 (PDF p9).

⑦ 한계와 변동 요인

  1. 2종 오류 (PDF p60): 부동층 ≈ 20% 가 막판에 결집하면 결과 변동 가능.
  2. 공표 금지 기간: 5/28 ~ 6/3 여론조사 공표 금지 → 그 사이 변화 미반영.
  3. 하우스 효과: 의뢰처에 따라 결과 편향. 추세가 일관되면 견고한 신호.
  4. 비응답 편향 (PDF p4): 응답률 10~12%, 무응답이 정치성향과 상관 시 편향.
  5. 지역 특성: 부산은 박빙(보수 텃밭에 야권 도전), 성남·서울은 정권심판 분위기.
