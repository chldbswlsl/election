## 1. 사용한 통계 개념과 공식

| 개념 | 공식 | PDF 슬라이드 |
|---|---|---|
| 모비율 95% 신뢰구간 | $\hat{p} \pm 1.96\sqrt{\hat{p}\hat{q}/n}$ | p44 |
| 오차범위(MoE) | $1.96 \cdot \sqrt{\hat{p}\hat{q}/n}$ | p44 |
| 두 후보 차이의 분산 (다항분포) | $\mathrm{Var}(\hat{p}_A-\hat{p}_B) = \dfrac{p_A + p_B - (p_A-p_B)^2}{n}$ | p44 확장 |
| 검정통계량 Z | $Z = \dfrac{\hat{p}_A-\hat{p}_B}{SE_\Delta}$ | p59 |
| p-value | $1-\Phi(Z)$, H1: $p_A > p_B$ | p59~60 |
| Poll-of-polls | 표본수 가중 평균 → 표준오차 $\propto 1/\sqrt{n_{\text{eff}}}$ | p10, p28 (CLT) |

> **왜 1.96 인가?** 표준정규분포 $Z$ 기준으로 양쪽 꼬리 합 5%를 잘라낼 때 $|Z|=1.96$. PDF p15 표준정규분포표에서 $P(0 \le Z \le 1.96) = 0.475$ → 양쪽 합쳐 0.95.

### 3-3. Poll-of-polls 합성 — 중심극한정리

#### A. 단순 합성 (n-가중, 29건 전체 — 5/27 최종)

표본수만 가중치로 사용. 유효 표본 $n_\text{eff}=28{,}242$ (7건 추가로 +6,198). 중심극한정리(PDF p28)에 의해 신뢰구간 폭이 단일 조사 대비 약 **1/6 수준**으로 좁아짐 (√(800/28,242) ≈ 0.168):

```
정원오 가중평균  : 46.62%   95% CI = [46.04%, 47.20%]
오세훈 가중평균  : 36.80%   95% CI = [36.24%, 37.36%]
차이             : +9.73%p   SE = 0.54%p
Z = +18.02       p-value ≈ 0 (< 10⁻¹⁶, 부동소수 한계)
```

> **22건(5/19) → 27건(5/26 중간) → 29건(5/27 최종) 변화**: 격차 +11.44 → +10.18 → **+9.73%p**. 약 1주 사이 −1.71%p 축소. 그러나 29건 단순 합산만으로도 Z=+18.02 → H0 기각 유지 (p < 10⁻¹⁶). 99% 신뢰수준에서 정원오 우세는 견고.

#### B. 가중 합성 (House Effect + 시간 감쇠, 대시보드와 동일)

538.com 식 메타분석. 의뢰처 편향(House Effect)과 *시간 감쇠*(반감기 30일)를 함께 반영:

- House Effect: 펜앤마이크/여론조사공정 = 0.5, 여론조사꽃 = 0.5, 그 외 = 1.0
- 시간 감쇠: $w_\text{time} = 0.5^{(\text{경과일수}/30)}$ — 30일 전 조사는 50%, 60일 전 조사는 25% 가중

가중 유효표본 $n_\text{eff}^\text{w} \approx 9{,}283$ (분석 기준일 2026-05-19):

```
정원오 가중평균  : 47.91%   95% CI = [46.90%, 48.93%]
오세훈 가중평균  : 36.18%   95% CI = [35.20%, 37.16%]
부동층(가중)     : 15.34%
차이             : +11.73%p   SE = 0.94%p
Z = +12.43       p-value ≈ 0 (< 10⁻¹⁶)
```

PDF p59 정의대로 **p-value = 1종 오류 확률 ≈ 0**. 99% 신뢰수준($Z=2.58$, PDF p47)에서도 H0를 확정적으로 기각.

#### C. 양자대결 vs 다자대결 분할 — 편향 점검

22건 중 양자대결 18건 / 다자대결 4건. 실제 6/3 선거는 다자 구도(개혁신당 김정철·정의당 권영국·여성의당 유지혜 등). 양자 폴이 표본수까지 압도(특히 5/12 여론조사꽃 양자 n=4,009, 4/15 여론조사꽃 자체조사 n=2,006)하면서 통합 격차가 양자 쪽으로 끌려갈 위험이 있어, 분할 결과를 함께 제시:

| 분할 | n | n_eff (단순) | 단순 격차 | 가중 격차 (House+Time) |
|---|---:|---:|---:|---:|
| **양자대결만 18건** | 18 | 18,841 | +12.36%p | **+13.58%p** |
| **다자대결만 4건** (= 후보확정 후, 5/10·5/11·5/14·5/17) | 4 | 3,203 | +6.03%p | **+6.52%p** |
| **통합 전체 22건** (기존 §A·§B) | 22 | 22,044 | +11.44%p | +11.73%p |

> 통합 합산(+11.73%p) 과 다자(+6.52%p) 사이 약 **5.2%p 격차**는 *양자대결 폴이 제3후보(약 1~3%) 표를 두 주자에게 나눠 흡수해 격차를 부풀린* 결과로 해석됨. 5/17 메트릭스 신규 폴 추가로 **다자 격차가 +8.10 → +6.52%p로 1.58%p 축소** — 5월 후반 박빙 신호가 다자 분할에 더 강하게 반영됨. 두 후보 비교의 통계적 유의성은 모든 분할에서 p ≪ 0.001 유지 (다자만 보아도 Z = +3.61).

단일 조사 n=800 과 합산 n_eff=22,000+ 의 분포를 비교하면 **분포의 폭(SE)이 약 1/5로** 줄어 두 후보 분포의 겹침 영역이 거의 사라집니다. 이것이 PDF p28 *중심극한정리* 의 시각적 의미 — $\sigma/\sqrt{n}$ 효과. (대시보드 ③ 카드에서 시각화)

PDF p59~60 의 *p-value* 정의 그대로 대시보드 ④ 카드에서 시각화: 표준정규분포에서 관측 $Z$ 위치가 임계값 $1.96$ (α=0.05) 와 $2.58$ (α=0.01) 을 한참 넘어선 위치 → 빨간 꼬리 면적(p-value)이 0에 가까움 → H0 기각이 매우 견고함.

## 5. 한계와 변동 요인 (PDF p2 "현실 데이터의 한계", p60 "오류")

1. **양자/다자 혼합 편향 (가장 중요)**: 20건 중 양자대결 16건이 표본수로 압도(특히 5/12 여론조사꽃 양자 n=4,009). 실제 선거는 다자 구도이므로 통합 합산(+11.70%p)은 양자 폴에 끌려간 *과장된 추정*. 다자 분리값 +8.05%p 가 모집단 추정으로 더 적절. §3-3 C 표·§4 의 두 시나리오 병렬 표시는 이 편향을 완화하기 위함.
2. **2종 오류 위험**: 부동층 약 16%가 실제 투표소에서 결집 시 결과 변동 가능. 다자 비관 시나리오(부동층 7:3 오세훈)는 격차가 +1.5%p로 좁아져 *오차범위 내 접전* 영역에 진입 (역전은 아님). 양자 기준은 비관에서도 +6.6%p 안전.
3. **공표 금지 기간**: 공직선거법 §108 에 따라 5/28 ~ 6/3 여론조사 공표 금지. 그 사이 변동을 본 분석은 포착하지 못함.
4. **하우스 효과(House Effect)**: 의뢰처에 따라 결과 편향. *대칭 디스카운트* (펜앤마이크/공정 0.5 ↔ 여론조사꽃 0.5) 적용 후 격차가 단순 합성 대비 −0.84%p 축소 (보수적 보정으로 작동). 보수성향 의뢰의 펜앤마이크조차 정원오 우세를 보여 → 추세는 견고.
5. **시간 감쇠의 임의성**: 30일 반감기는 538.com 등이 사용하는 표준값이지만 *정량 검증된 값은 아님*. 반감기 14일 → 격차 약 +13%p, 60일 → 약 +10.5%p 로 변동 가능.
6. **표본 ≠ 모집단**: 응답률 5~12% (PDF p4 *비정형 데이터의 한계*). 무응답이 정치성향과 상관 시 편향. 다자 폴 응답률(평균 ~7%)이 양자 폴 응답률(~7%)과 큰 차이 없어 응답률 자체에서 양자/다자 격차 차이를 설명하긴 어려움.
7. **베이지안 사전(prior) 보정 미반영**: PDF p52~53 *Bayesian* 개념을 적용하면 과거 서울시장 여당 우세 사전, 2025 대선 결과 등으로 사후 확률을 더 보수적으로 조정 가능.

| 2026-04-20 펜앤마이크 (정 49.5/오 30.9) | 펜앤마이크 아카이브 + site:검색 + 매체별 검색 모두 해당 수치 미발견 |

이 두 건이 제거되어도 결론에 본질적 영향 없음 (제거 전 +11.70%p → 보정 후 19건 +11.61%p → **신규 3건 추가 후 22건 +11.73%p**).

#### 5월 후반 박빙 신호 — 추세 변화 분석

| 시점 | 다자 폴 | 양자 폴 | 다자 평균 격차 |
|---|---|---|---:|
| 4월 후반 (4/11~28) | (다자 없음) | 4건 모두 +10~26%p | — |
| 5월 초~중 (5/2~14) | 3건 (+2~11%p, 평균 +7%p) | 6건 (+7~21%p, 평균 +14%p) | +8.10%p |
| 5월 후반 (5/13~17) | **5/17 메트릭스 +3%p** ⚠️ | (5/13 KSOI +5.1%p) | **+6.52%p** |

**해석**: 다자대결에서 4월 시점엔 폴 자체가 부재했고, 5월 들어 박빙 결과가 누적되고 있음. 5/17 메트릭스의 +3%p는 보수성향 의뢰처가 아닌 *조선일보·메트릭스* 정통 폴이라는 점에서 House Effect로 깎기 어려운 신호.
